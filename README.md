# MCCP

A Python 2.7.9 Minecraft server manager

Linux only

Current internal commands:
---------------------------
* /start
* /status
* /stop (overrides stop)
* /restart
* /quit

Very WIP.

Installing
---------------
* `sudo pip install twisted`
* `sudo pip install simplejson`
* `git clone https://github.com/drazisil/mccp.git`
